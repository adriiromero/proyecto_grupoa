# 🛠️ Proyecto TecnoBalanegra - Web Corporativa

Este repositorio contiene el código fuente de la plataforma web de **TecnoBalanegra**, desarrollada como parte del Proyecto Final del Grado Medio en Sistemas Microinformáticos y Redes (SMR) del IES Abdera en el curso 2025/26.

## Sobre el Proyecto
TecnoBalanegra es una empresa de servicios informáticos híbridos (físicos y digitales) ubicada en Balanegra, Almería.
*   Esta es una **Web Estática**, diseñada para ofrecer **información corporativa, servicios de taller y contacto.**

## Tecnologías Utilizadas
*   **Lenguajes:** HTML y CSS para la estructura y el diseño visual.
*   **Gestión de Versiones:** Git y GitHub para el control del código fuente.
*   **Entorno de Desarrollo:** Visual Studio Code.

## Estructura del Repositorio
*   `/index.html`: Página principal de acceso.
*   `/css/`: Hojas de estilo para el diseño responsive.
*   `/img/`: Recursos gráficos y logotipos de la empresa.

## Instrucciones para la Revisión Técnica
Para ejecutar y revisar este proyecto en un entorno local, siga estos pasos:

1.  **Descarga:** Pulse el botón verde `<> Code` y seleccione `Download ZIP`.
2.  **Extracción:** Descomprima el archivo en su equipo.
3.  **Visualización:** 
    *   Abra la carpeta en **Visual Studio Code**.
    *   Haga clic derecho sobre `index.html` y seleccione `Open with Live Server` o simplemente abra el archivo en cualquier navegador.
